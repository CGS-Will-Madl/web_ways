HTML Lesson 3 - Images

To enbed a photo in your web page just use the code "<img src="learning_html.jpg" alt="Learning HTML">". Where it says src="learning_html.jpg" that is where you put the name of the image or the URL of the image. alt="Learning HTML" is what you wish to name the image if the image does not display in the case of an error.

If you wish to control the height and width of the image to adjust the size, use the code "<img src="html.png" alt="HTML 5 Logo" style="height:640px; width:800px;">". You can make the dimensions whatever you wish.
