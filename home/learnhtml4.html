HTML Lesson 4 - Tables

To create a table you need 4 main tags. "<table>" "<tr>" "<th>" "<td>"

"<table>" is what you use to start off the table. It tells the browser that the code following is to be put in a table.

"<tr>" is the row, use this to create a row

"<th>" is the table column header

"<td>" is what is in the cell, the data held inside the cell. Note that they can contain all sorts of HTML elements such as text, images, lists, other tables, etc.
