Lesson 1 - Introduction

CSS is crucial to the look of a website.
It can make the difference of having an average website and a professional website.
You link css by typing <link type="text/css" rel="stylesheet" href="[filename].css" media="screen"> into the head tag of your HTML.

You can also type css directly into your HTML code by using the <style> tag.

Once you get the hang of CSS it gets easier and easier.
